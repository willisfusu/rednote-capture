# Font Files

This directory should contain the NotoSansSC font for Chinese character rendering in PDFs.

## Required Font

**NotoSansSC-Regular.ttf** - Google Noto Sans Simplified Chinese

## Download Instructions

1. Visit: https://fonts.google.com/noto/specimen/Noto+Sans+SC
2. Download the Regular weight
3. Place the .ttf file in this directory
4. Rename to `NotoSansSC-Regular.ttf`

Alternatively, use a subset font generator to create a smaller file (~2-3MB instead of ~16MB):
- https://www.fontsquirrel.com/tools/webfont-generator
- Select only the characters you need (common CJK range)
